Colors

Having the correct brand colors available when starting a new project is a must. Whats also important is how you define those brand color variables. Try to stay away from color specific and non semantic variables. If the client/customer decides to change a color at any given point you can easily achieve the update by changing the color hex-code vs changing the variable name across several sass files.
